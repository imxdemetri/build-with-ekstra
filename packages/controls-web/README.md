# `@ekstra/controls-web`

Reference web control profiles for Ekstra-based browser experiences.

Initial profiles:
- `pointerBasic`
- `pointerPrecision`
- `presentationRemote`
- `orbit3d`
- `kioskNav`
- `mediaRemote`
- `accessibilityNav`
- `docScroll`
- `modelInspect`
- `relaxedPointer`

This package defines browser-facing control profile metadata and defaults.
It does not replace the Ekstra runtime or browser bridge.
