# Changelog

This changelog tracks changes to the public `build-with-ekstra` surface.

It is not a full history of the private Ekstra runtime.

## 0.1.0 - 2026-03-07

First public repository release for `build-with-ekstra`.

- Apache-2.0 license published at the repo root
- hosted developer surface under `ekstra.ai/build-with-ekstra`
- two supported public starters:
  - `web-phone-pointer`
  - `presentation-remote`
- same-domain hosted controller routes for pointer and presentation flows
- public browser contract, support-status docs, wiki, and launch board
- published npm preview package `@ekstraai/controls-web@0.1.0-preview.2`
- self-hosted Docker manifests published with preview-grade runtime image guidance

## 0.1.0-preview.1 - 2026-03-06

Initial public developer preview baseline:

- public developer landing and docs
- hosted sandbox under `ekstra.ai`
- `web-phone-pointer` as the primary supported public starter
- `@ekstraai/controls-web` package scaffold and public profile catalog
- GitHub wiki, support surface, and public launch board

## Unreleased

Current public hardening work:

- explicit public contract documentation
- supported versus preview status matrix
- tighter security and hosted sandbox expectations
- stronger package metadata and control profile support semantics
- `presentation-remote` as the second public starter
- first published npm preview release for `@ekstraai/controls-web`
